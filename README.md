# 📊 Interactive Excel Dashboard

An interactive Microsoft Excel Dashboard designed to transform raw data into meaningful insights using charts, pivot tables, slicers, formulas, and automation.

The dashboard enables users to monitor KPIs, analyze trends, and generate reports through a clean and user-friendly interface.
